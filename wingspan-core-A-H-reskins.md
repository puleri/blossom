# Wingspan Core Set (A–H) Plant Reskins

Reference lane mapping used: **Forest → Root (Understory)**, **Wetlands → Pollinate (Meadow)**, **Grasslands → To the Sun (Desert)**, **Eggs → Sunlight**, **Tuck/Cache → Compost**.

## A
- **Acorn Woodpecker → Dune Cache Oak**: Understory-only seed-storer; on Root activation gain **1 seed or 1 compost**.
- **American Avocet → Saltflat Mirrorleaf**: Expensive point bloom; bonus if another plant has a basin nest-equivalent trait.
- **American Bittern → Fen Reed Sentinel**: High raw points, tiny sunlight cap, minimal engine value.
- **American Coot → Marsh Reed Collector**: Pollinate draw+compost engine with large sunlight storage.
- **American Crow → Ashgrove Forager**: Tri-biome converter; spend sunlight to gain 2 mixed resources (weaker Raven line).
- **American Goldfinch → Sandgrain Finchbloom**: Refund-style seed plant; weak if paid full cost.
- **American Kestrel → Dust Talon Succulent**: Predator-tagged point plant; unreliable hunt effect.
- **American Oystercatcher → Tideline Shellroot**: Selective tray-draw support that can feed opponents.
- **American Redstart → Embertail Nectar Sprig**: Understory food accelerator without table sharing.
- **American Robin → Thicket Redberry**: Non-meadow compost/draw line; decent emergency sunlight bank.
- **American White Pelican → Shoal Net Lotus**: Fish-linked compost scorer that spikes with direct fish support.
- **American Woodcock → Hollow Fern Monument**: Pure efficient point bomb.
- **Anhinga → Bog Spearvine**: Mediocre frame, predator-synergy point bomb when supported.
- **Anna’s Hummingbird → Scarlet Nectar Bloom**: Tri-biome flexible accelerator; gain any 1 root resource.
- **Ash-Throated Flycatcher → Drywind Cupvine**: Late-game burst points based on matching nest-structure plants.
- **Atlantic Puffin → Starpool Driftbloom**: High-variance mega point bomb tied to lucky reveals.

## B
- **Baird’s Sparrow → Prairie Threadgrass**: Useful lay-anywhere sunlight power but overcosted.
- **Bald Eagle → Apex Sun Raptor Bloom**: Expensive predator bomb with occasional fish refund.
- **Baltimore Oriole → Amberfruit Bannerflower**: Point bomb with occasional fruit smoothing.
- **Barn Owl → Loftshade Owlroot**: Solid body, weak power; bonus-card dependent bomb.
- **Barn Swallow → Eaveswift Petal**: Cheap Pollinate compost/draw micro-engine.
- **Barred Owl → Stripebark Nightbloom**: Weak baseline, rescued by predator bonus tags.
- **Barrow’s Goldeneye → Coldpool Mirrorreed**: Overpriced mid-point body needing nest support partner.
- **Bell’s Vireo → Bellvine Archivist**: Classic bonus-card seeker with star-structure flexibility.
- **Belted Kingfisher → Riverbelt Spearleaf**: Good sunlight bank; minor pink fish trickle.
- **Bewick’s Wren → Brushline Migrant Bloom**: Overpriced migrator, generally inefficient.
- **Black Skimmer → Nightblade Surfaceleaf**: Predator-tagged point bomb with weak action power.
- **Black Tern → Sootwing Marsh Petal**: Cheap cycle/look effect with decent star capacity.
- **Black Vulture → Carrion Draft Root**: Zero-cost reactive scavenger; only good in predator-heavy tables.
- **Black-Bellied Whistling Duck → Emberwhistle Basin Reed**: Strong sunlight bank that scales with seed support.
- **Black-Billed Magpie → Obsidian Trinketvine**: Better scavenger profile, wild cost, utility tags.
- **Black-Chinned Hummingbird → Onyx Nectar Bloom**: Tri-biome fixer; weaker hummingbird payoff.
- **Black-Crowned Night-Heron → Crowned Mire Harvester**: Premium Crow-line converter, strong bomb profile.
- **Black-Necked Stilt → Saltmarsh Stiltreed**: Outclassed draw piece; only niche bonus-tag value.
- **Blue Grosbeak → Sapphire Prairie Migrant**: Overpriced migrator with weak return.
- **Blue Jay → Azure Cachebloom**: Conditional feeder-dependent gain-or-compost choice.
- **Blue-Gray Gnatcatcher → Mistthread Gnatflower**: Cheap early fixer; efficient tempo stabilizer.
- **Blue-Winged Warbler → Cerulean Flashbloom**: Undercosted high-point burst plant.
- **Bobolink → Meadow Chimegrass**: Nest-setup dependent late-game point burst.
- **Brant → Brinefront Planner Reed**: Weak frame, but strong timing-based tray filtering.
- **Brewer’s Blackbird → Charmeadow Tuckgrass**: Grasslands-locked compost/lay archetype.
- **Broad Winged Hawk → Broadsun Talonleaf**: Cheap predator-tagged body with weak hunt power.
- **Bronzed Cowbird → Bronze Broodvine**: Great with early cup-structure accelerators.
- **Brown Pelican → Siltbeak Basin Lotus**: Fish-refund illusion card; low standalone quality.
- **Brown-Headed Cowbird → Rustcap Broodvine**: Low-cost sunlight parasitic support, highly efficient.
- **Burrowing Owl → Burrow Halo Cactus**: Good points+capacity, weak power, predator bonus friendly.
- **Bushtit → Brushtail Latticevine**: Tri-biome compost/lay with large star sunlight cap.

## C
- **California Condor → Condorshade Scavenger Bloom**: Zero-cost bonus hunter with very poor base stats.
- **California Quail → Chaparral Quillgrass**: Classic forest-to-sunlight layer with solid capacity.
- **Canada Goose → Northwind Seedpacker**: Discard 1 seed to compost 2 cards, support-dependent scorer.
- **Canvasback → Canvas Reed Drifter**: Flexible early Pollinate acceleration with large sunlight bank.
- **Carolina Chickadee → Pinechip Cachebud**: Cheap caching line that must be played early.
- **Carolina Wren → Hearthwren Cardbloom**: Excellent cheap early draw acceleration and egg-bank analog.
- **Cassin’s Finch → Cassin Archive Petal**: Reliable bonus-card archetype piece.
- **Cassin’s Sparrow → Cassin Sunspike Grass**: Overpriced grassland sunlight spreader.
- **Cedar Waxwing → Cedar Gleam Berryvine**: Strong if supported despite no Pollinate placement.
- **Cerulean Warbler → Cerulean Chronicle Bloom**: Pure bonus-card value plant.
- **Chestnut-Collared Longspur → Chestnut Ledgergrass**: High-cost/high-point bonus archetype variant.
- **Chihuahuan Raven → Chihuahuan Ash-Raven Vine**: Top-tier sunlight-to-resource converter; metagame-defining.
- **Chimney Swift → Chimney Spiralpetal**: Migrator with broad goal tags but low impact capacity.
- **Chipping Sparrow → Chipping Sunthread**: Elite forest-side sunlight spammer.
- **Clark’s Grebe → Clark’s Basin Glideroot**: Extreme points-per-cost efficiency with neutral draw exchange.
- **Clark’s Nutcracker → Clark’s Vaultcone**: Conditional but flexible point bomb.
- **Common Grackle → Common Gritgrackle Vine**: Tri-biome compost/lay engine, tag-scaling scorer.
- **Common Loon → Common Deepcall Reed**: Conditional converter, good with omnivore-style bonuses.
- **Common Merganser → Common Stream Mergroot**: Inconsistent hunt effect on good point shell.
- **Common Nighthawk → Common Dusk Migrant Bloom**: Mediocre migrator despite bonus-tag help.
- **Common Raven → Common Root-Raven Vine**: Premier sunlight-to-resource engine, especially in Desert lane.
- **Common Yellowthroat → Common Goldthroat Reed**: One of the best early high-volume draw engines.
- **Cooper’s Hawk → Cooper’s Talon Thornbloom**: Consistent predator line with strong bonus scaling.

## D
- **Dark-Eyed Junco → Duskeye Echoreed**: Mini repeat engine constrained by no Pollinate access.
- **Dickcissel → Prairie Cuttergrass**: Overpriced grassland-only compost/lay piece.
- **Double-Crested Cormorant → Twincrest Tide Cormroot**: Fish-fed discard-to-double-compost scorer.
- **Downy Woodpecker → Downy Graftpecker Tree**: Cheap “play another plant” setup card.

## E
- **Eastern Bluebird → Eastern Skyberry Shrub**: Strong “play another plant” with big sunlight capacity.
- **Eastern Kingbird → Eastern Crownking Bloom**: Low-impact pink trigger; weak overall profile.
- **Eastern Phoebe → Eastern Field Phoebloom**: Tri-biome early resource accelerator that shares insects.
- **Eastern Screech Owl → Eastern Screech Thornowl**: Efficient predator-tag point bomb, low consistency power.

## F
- **Ferruginous Hawk → Ferric Dune Hawkthorn**: Predator-tag bomb with low consistency hunt.
- **Fish Crow → Brackish Fishcrow Vine**: Half-Raven converter; decent Desert resource support.
- **Forster’s Tern → Forster’s Marsh Ternpetal**: Cheap look-and-cycle card that breaks even on draw.
- **Franklin’s Gull → Franklin’s Scrapgull Bloom**: Elite sunlight-to-cards converter central to egg-spam analogs.

## G
- **Golden Eagle → Gilded Apex Eaglethorn**: Expensive predator bomb with top-end conversion consistency.
- **Grasshopper Sparrow → Hoppergrass Sunlayer**: Weak base saved by lay-on-any-plant sunlight action.
- **Gray Catbird → Gray Echo Catvine**: Expensive repeat card that doubles best engine activations.
- **Great Blue Heron → Great Azure Heronroot**: Beefy “play another plant” archetype body.
- **Great Crested Flycatcher → Great Crest Flyvine**: Conditional feeder accelerator with solid midpoint stats.
- **Great Egret → Great Egret Monument Reed**: Premium high-point “play another plant” finisher.
- **Great Horned Owl → Great Hornthorn Owlbloom**: Expensive predator bomb with high consistency ceiling.
- **Greater Prairie-Chicken → Greater Prairie Ledgergrass**: Heavy bonus-card archetype with bigger capacity.
- **Greater Roadrunner → Greater Roadthorn Runnervine**: Expensive predator bomb with low consistency.
- **Green Heron → Green Filter Heronleaf**: Flexible food-filtering utility; strong with nectar systems.

## H
- **Hermit Thrush → Hermit Shade Thrushbloom**: Expensive point bomb with situational majority/minority pink power.
- **Hooded Merganser → Hooded Mire Repeater**: Wetlands-only predator repeater with strong expansion upside.
- **Hooded Warbler → Hooded Goldwarbler Bloom**: Efficient 7-point, 2-cost point-heavy body.
- **Horned Lark → Horned Bluff Larkgrass**: Conditional opponent-driven pink tuck from hand.
- **House Finch → Homestead Finchvine**: Tri-biome compost/draw with huge sunlight bank.
- **House Wren → Homestead Wrenwood**: Dual-biome “play another plant” with strong sunlight storage.
